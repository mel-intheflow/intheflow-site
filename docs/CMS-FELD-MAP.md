# CMS Feld-Map (InTheFlow)

Diese Datei zeigt: **CMS-Feld → sichtbarer Bereich auf der Website**.

## Brand
- `brand.subtitle` → Untertitel neben Logo im Header

## Hero
- `hero.headline` → große Hauptüberschrift oben
- `hero.text1` → erster Intro-Absatz
- `hero.text2` → zweiter Intro-Absatz
- `hero.ctaPrimaryLabel` → Text Primary-Button (Kurse)
- `hero.ctaSecondaryLabel` → Text Secondary-Link (Kontakt)
- `hero.badgeTitle` → Titel im Badge rechts beim Hero-Bild
- `hero.badgeText` → Badge-Text rechts beim Hero-Bild

## About / Werdegang
- `about.leftHeading` → Headline linke Werdegang-Spalte
- `about.leftParagraphs[]` → Werdegang-Absätze links
- `about.rightHeading` → Headline rechte Zertifikate-Spalte
- `about.infoNote` → Hinweisbox unter Zertifikaten

## Ganzheitlicher Ansatz
- `approach.heading` → Section-Headline
- `approach.intro` → Intro-Text
- `approach.zielgruppeTitle` → Titel Karte „Zielgruppe"
- `approach.zielgruppeText1` → Zielgruppe Text 1
- `approach.zielgruppeText2` → Zielgruppe Text 2
- `approach.arbeitsweiseTitle` → Titel Karte „Arbeitsweise"
- `approach.arbeitsweiseText1` → Arbeitsweise Text 1
- `approach.arbeitsweiseText2` → Arbeitsweise Text 2
- `approach.fastenTitle` → Titel Karte „Fastenbegleitung"
- `approach.fastenText1` → Fasten Text 1
- `approach.fastenText2` → Fasten Text 2

## Angebot
- `angebot.headline` → Headline „Angebot"
- `angebot.intro` → Introtext „Angebot"

### Angebotskarten
- `offerCards[].kicker` → Karten-Kicker (z. B. Yoga/Fasten)
- `offerCards[].title` → Karten-Titel
- `offerCards[].items[]` → Bullet-Liste je Karte

### Hinweiskarte
- `hinweis.title` → Titel Hinweiskarte
- `hinweis.text1` → Hinweistext 1
- `hinweis.text2` → Hinweistext 2

## Kurse
- `kurse.headline` → Headline Kurs-Section
- `kurse.intro` → Intro Kurs-Section

### Kurskarten
- `courseCards[].kicker` → Karten-Kicker
- `courseCards[].title` → Karten-Titel
- `courseCards[].rows[]` → Zeilen in der Kartenliste (Ort/Zeit/Info etc.)

### Schnuppern
- `schnuppern.title` → Titel Schnuppern-Karte
- `schnuppern.text` → Text Schnuppern-Karte

## Veranstaltungen
- `veranstaltungen.headline` → Headline Veranstaltungs-Section
- `veranstaltungen.intro` → Intro Veranstaltungs-Section

### Event-Karten
- `events[].kicker` → Event-Kicker
- `events[].title` → Event-Titel
- `events[].text` → Event-Beschreibung
- `events[].format` → Meta-Zeile „Format"
- `events[].haeufigkeit` → Meta-Zeile „Häufigkeit"

## Kontakt
- `kontakt.headline` → Kontakt-Headline
- `kontakt.intro` → Kontakt-Intro
- `kontakt.whatsappLabel` → WhatsApp-Buttontext
- `kontakt.quote` → Zitat in rechter Karte
- `kontakt.quoteText` → Zusatztext unter Zitat
- `kontakt.statusTitle` → Titel Karte „Freie Plätze und Einstieg"
- `kontakt.statusText1` → Status Text 1
- `kontakt.statusText2` → Status Text 2
- `kontakt.impulsTitle` → Titel Karte „Kontaktimpuls"
- `kontakt.impulsText1` → Impuls Text 1
- `kontakt.impulsText2` → Impuls Text 2

## Footer
- `footer.copyright` → Footer Copyright-Text
